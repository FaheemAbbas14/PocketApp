## Before You Upload

### Create GitHub Repo
1. Go to https://github.com/new
2. Name it: `pocket-app`
3. Click "Create repository"

### Get Access Token
1. Go to https://github.com/settings/tokens
2. Generate new token (classic)
3. Check "repo" scope
4. Copy token (use as password when uploading)
